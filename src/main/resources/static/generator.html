Comment fonctionne le générateur

Le générateur utilise le modèle de prédiction pour créer du texte à partir d'un préfixe donné.

Voici le processus de génération :

  1. Vous fournissez un préfixe qui servira de point de départ
  2. L'application analyse ce préfixe selon la taille de n-gramme choisie
  3. Pour chaque mot à générer, l'application :
    • Recherche les mots qui pourraient suivre la séquence actuelle
    • Sélectionne un mot aléatoirement en fonction des probabilités
    • Ajoute ce mot au texte généré
    • Utilise la nouvelle séquence pour le mot suivant

Le résultat est un texte qui imite statistiquement le style des textes utilisés pour l'entraînement.
