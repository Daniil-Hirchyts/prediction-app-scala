Comment fonctionnent les suggestions

Les suggestions sont basées sur l'analyse statistique des mots qui apparaissent fréquemment ensemble dans les textes utilisés pour entraîner le modèle.

Voici comment l'outil fonctionne :

  1. Le texte que vous entrez est analysé pour identifier les derniers mots (selon la taille du n-gramme choisie)
  2. L'application recherche dans le modèle quels mots suivent le plus souvent cette séquence
  3. Les mots les plus probables sont présentés comme suggestions, avec leur pourcentage de probabilité

Vous pouvez cliquer sur l'une des suggestions pour l'ajouter à votre texte.
